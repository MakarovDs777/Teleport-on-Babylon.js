# Teleport-on-Babylon.js

Классическое табло телепортации включающее в себя XYZ координаты поля для переключения на от 0 до бесконечности не включая бесконечность.

[Teleport with a coordinate board | Babylon.js Playground](https://playground.babylonjs.com/#W4MF28#0)

Табло телепортации с возможностью телепортироваться случайно в пределах от мнимального до максимального значения XYZ на от 0 до бесконечности не включая бесконечность.

[Teleport with random movement to a random range coordinate, and an automatic movement button to a random coordinate | Babylon.js Playground](https://playground.babylonjs.com/#4L4RFI#0)
